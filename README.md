# Twitter-sentiment-analysis


A simple sentiment analysis tool that predicts whether text or tweets are Positive or Negative.
The project uses a trained ML model (TF-IDF + classifier) and a Streamlit interface.

Included Files

app.py – Main Streamlit application

Twitter_sentiment_Analysis.ipynb – Model training notebook

model.pkl – Trained model

vectorizer.pkl – TF-IDF vectorizer



Run the App

streamlit run app.py



Features

Analyze custom text

Fetch and analyze tweets via Nitter

Clean UI with colored sentiment cards
